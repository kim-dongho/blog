# LRU Cache Solution in JavaScript

이 코드는 **LRU (Least Recently Used)** 알고리즘을 사용하여 캐시 동작을 시뮬레이션합니다. `cacheSize`는 캐시의 크기를 정의하고, `cities`는 사용자가 방문한 도시의 목록을 나타냅니다. 각 도시에 대해 캐시 적중 여부에 따라 시간을 추가합니다.

```jsx
function solution(cacheSize, cities) {
  var answer = 0;
  let arr = [];

  if (cacheSize === 0) return cities.length * 5;

  for (let city of cities) {
    city = city.toUpperCase();

    if (arr.includes(city)) {
      let index = arr.indexOf(city);
      arr.splice(index, 1);
      arr.unshift(city);
      answer += 1;
    } else {
      if (!(arr.length < cacheSize)) {
        arr.pop();
      }
      arr.unshift(city);
      answer += 5;
    }
  }
  return answer;
}
```
